# 无人机保险 AI 核保引擎 🚁

晶世科保 — 2周MVP原型

## 功能

- **AI风险评分**: 7维度评分卡（设备/场景/环境/资质/暴露量/理赔/保额）
- **智能定价**: 基于风险评分的费率区间建议
- **数据飞轮**: 询价→报价→成交/拒保全链路记录
- **缺失检测**: 自动识别并追问缺失信息

## 技术栈

- Streamlit (UI)
- Python 3.14
- pandas + numpy
